Yuki
====
Analyze all the videos in a folder,
export the thumbnails and output the Excel file.

![icon](https://github.com/loonghao/Yuki/blob/master/src/yuki/resources/images/yuki.png)

Features:
- Analyze videos.
- Export thumbnails.
- Save to Excel.
